## 🚀 **How to Get Started**

### 1. **Install Dependencies**
Run one of the setup scripts to install all required packages:

**Windows:**
```cmd
setup.bat
```

**macOS/Linux:**
```bash
chmod +x setup.sh && ./setup.sh
```

**Manual:**
```bash
npm install
```

### 2. **Start Development**
```bash
npm run dev
```

### 3. **Build for Production**
```bash
npm run build
npm run dist
```
